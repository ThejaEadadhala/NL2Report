# TPC-H Data Processing Project
**Week 3 — Real TPC-H Dataset, PostgreSQL Backend**

## Project structure

```
NL2Report/
  ├── raw/                  # TPC-H CSVs generated by generate_data.py
  ├── processed/            # cleaned CSVs output by tpch_pipeline.py
  ├── venv/                 # Python virtual environment
  ├── generate_data.py      # STEP 1 — generates real TPC-H SF=1 data
  ├── tpch_pipeline.py      # STEP 2 — cleans data and loads into PostgreSQL
  ├── tpch_filter.py        # STEP 3 — query/filter data from PostgreSQL
  ├── schema.sql            # PostgreSQL table definitions
  ├── constraints.sql       # primary keys, foreign keys, indexes
  ├── requirements.txt      # Python dependencies
  └── README.md
```

---

## Setup order

### 1. Activate virtual environment
```bash
venv\Scripts\activate
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Generate TPC-H data
```bash
python generate_data.py
```
Exports 8 CSV files to `raw/` (~1GB total, 6M lineitem rows)

### 4. Create PostgreSQL database and schema
```bash
psql -U postgres -c "CREATE DATABASE tpch;"
psql -U postgres -d tpch -f schema.sql
```

### 5. Run the pipeline
```bash
python tpch_pipeline.py
```
Reads from `raw/` → cleans → saves to `processed/` → loads into PostgreSQL

### 6. Apply constraints
```bash
psql -U postgres -d tpch -f constraints.sql
```

### 7. Query the data
```bash
python tpch_filter.py
```
